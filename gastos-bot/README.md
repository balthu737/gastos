# Chat bot para dinero
## Qué hace? para qué sive? 
Este chat bot sirve para llevar las cuentas de tu día a día, la idea nacio para hacer facil el conteo de dineo, ya sea en gastos, ahorros, inversiones (todavia esta en una verison temprana del desarrollo)
## Como instalar 
Para instalar correctamente: 
Copia esto en la raiz del proyecto y instala
1. pip install requeriments.txt
2. python main.py
## Comandos
Los comando que el bot tiene son los sigientes 
### /start
### / help
### / dia
### / semana
### /anio
